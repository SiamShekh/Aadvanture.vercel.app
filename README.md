## Run Locally

Clone the project

```bash
  git clone https://github.com/SiamShekh/Aadvanture.vercel.app.git
```

Go to the project directory

```bash
  cd Aadvanture.vercel.app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
